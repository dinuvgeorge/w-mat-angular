Wrappers Angular Material

Custom Components for Angular using Material Design

Wrappers Angular Material UI Components demos

W-Mat for Angular is a modern Angular UI Components library that has been built from the ground up to be lightweight, responsive, modular and touch friendly. It is written in TypeScript and Angular Material. It also includes complete support for Angular framework.


Explore and learn using feature-wise demos in Components section and example applications in Showcase applications section.
